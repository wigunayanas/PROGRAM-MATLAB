Program Registrasi SIM 
1. untuk memulai program, jalankan / run file dashboard.m -> akan muncul tampilan awal dan bisa di klik mulai.
2. setelahnya akan redirect ke page selanjutnya yaitu data permohonan.
3. pada page data permohonan, silahkan isi data sesuai dengan data diri anda agar dapat diproses oleh program.
4. pada folder ini, disediakan 4 sampel foto. apabila ingin menggunakan foto sendiri boleh :v tinggal klik browse.
5. outputnya akan muncul pada bagian sidebar kanan dengan judul form registrasi.
6. Program akan merekam serta memunculkan kembali data diri anda yang telah di isi pada form.
